## 🚀 Getting Started

To run this project locally:

```bash
# Clone the repo
git clone https://github.com/kamalkori28/MyPortfolio.git

# Navigate to the folder
cd MyPortfolio

# Install dependencies
npm install

# Start the development server
npm run dev
